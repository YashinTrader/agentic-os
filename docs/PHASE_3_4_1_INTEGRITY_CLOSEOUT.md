## F1 — Test artifact invariant

`runtime/unittest_last_run.txt` is generated by `scripts/run_tests.py`, never hand-edited.

Required invariant:

```
artifact.commit_full == tests_commit_sha == implementation_sha
```

The artifact commit may be **later** than `implementation_sha` because committing the artifact advances HEAD. The artifact still records the **implementation** commit where tests ran, not the artifact commit itself.

## F3 — Artifact cross-check and validator-at-HEAD

`scripts/verify_repository_verification.py` now:

1. Parses `runtime/unittest_last_run.txt`
2. Compares `commit_full`, `test_count`, `exit_code`, and `repo_root` against the handoff block
3. Confirms artifact commit is an ancestor of branch HEAD
4. Runs `sys.executable scripts/validate.py` at actual checkout (`shell=False`, bounded timeout)
5. Requires validator exit 0 and compares with declared `validator_exit_code`

This proves **repository integrity**, not code correctness by itself. Claude still independently reruns the full suite before approval.

This is **not** cryptographic or CI-grade attestation.
